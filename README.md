# LeilaoIngressos - SDK JS
SDK js para utilização da API da leilaoingressos.com

## Instalação

Para instalar o sdk, siga abaixo as instruções:

```bash
$ npm install leilao-sdk --save # para instalar e adicionar no pacote package.json
```
